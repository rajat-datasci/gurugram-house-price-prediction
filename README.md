# 📊 Gurugram House Price Prediction



## 📌 Overview

This project focuses on building a machine learning model to predict house prices based on various features.  

Since a clean Gurugram dataset was not available, the California Housing Dataset [https://www.kaggle.com/datasets/camnugent/california-housing-prices] was used as a proxy to simulate real-world scenarios.



## 🎯 Objective

- Understand how different factors influence house prices  

- Build a predictive model for price estimation  

- Compare multiple regression algorithms  

- Create a reusable and scalable ML pipeline  



## 🛠️ Tech Stack

- Language: Python  

- Libraries: NumPy, Pandas, Matplotlib, Scikit-Learn  

- Tools: VS Code, Jupyter Notebook  

- Model Persistence: Joblib
